Lab 13 – Mobile Navigation

Overview

В данной лабораторной работе реализована навигация в приложении React Native с использованием библиотеки React Navigation. Были изучены и применены основные подходы навигации: Stack Navigation и Bottom Tab Navigation.  ￼

⸻
 Task 1 – Stack Navigation

Что сделано:
	•	Настроен NavigationContainer
	•	Реализован Stack.Navigator
	•	Созданы экраны:
	•	Home
	•	Profile
	•	Settings
	•	Добавлена навигация между экранами (navigation.navigate)
	•	Передача параметров (userId)
	•	Настроен header

Результат:

Реализована последовательная (иерархическая) навигация между экранами приложения.

⸻

Task 2 – Bottom Tab Navigation

Что сделано:
	•	Реализован Tab.Navigator
	•	Добавлены вкладки:
	•	Home
	•	Search
	•	Notifications
	•	Profile
	•	Добавлены иконки (@expo/vector-icons)
	•	Реализован badge для уведомлений
	•	Скомбинированы Stack + Tab навигаторы

Результат:

Пользователь может быстро переключаться между основными разделами приложения.

⸻

Использованные технологии
	•	React Native
	•	TypeScript
	•	React Navigation
	•	Expo
