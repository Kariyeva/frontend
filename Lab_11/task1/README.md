## React Native vs React (Web)
React for web uses DOM elements such as `div`, `span`, and `p`.  
React Native uses native mobile components such as `View`, `Text`, `Image`, and `TextInput`.

### Main differences
- Web React renders to the DOM
- React Native renders to native mobile components
- Web React uses CSS / className
- React Native uses `StyleSheet.create()`
- Web React often uses `onClick`
- React Native uses `onPress`
