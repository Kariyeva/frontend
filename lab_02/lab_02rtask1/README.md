# Lab 02 — Task 1 (Fragments & List Mapping)

## Описание
В этой лабораторной работе были изучены основы JSX, использование фрагментов и отображение списков с помощью метода map().

## Использованные технологии
- React
- TypeScript
- JSX
- Fragments

## Реализованный функционал

### FragmentLayout
Создан компонент, который возвращает три элемента:
- header
- main
- footer

Эти элементы объединены с помощью React fragments (<>...</>), без использования лишнего wrapper div.

### ItemList
Создан список элементов, который отображается через метод map().

Массив содержит несколько объектов, например:
{ id: 1, name: "Apple" }
{ id: 2, name: "Banana" }
{ id: 3, name: "Cherry" }

Каждый элемент отображается как `<li>` внутри `<ul>`.

Для каждого элемента используется уникальный `key`.

### Combined Component
Создан компонент, который возвращает:
- заголовок
- список элементов
- текст с количеством элементов

Количество элементов выводится через выражение:
items.length

## Вывод
В ходе лабораторной работы были изучены:
- использование fragments
- JSX expressions
- отображение списков через map()
