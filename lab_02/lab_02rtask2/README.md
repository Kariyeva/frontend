# Lab 02 — Task 2 (Custom Components & Dynamic Props)

## Описание
В этой лабораторной работе были реализованы собственные React компоненты и передача данных через props.

## Использованные технологии
- React
- TypeScript
- JSX
- Props
- Fragments

## Реализованные компоненты

### Card
Создан переиспользуемый компонент Card.

Компонент принимает props:
- title
- children
- className (опционально)

Компонент отображает:
- заголовок
- содержимое карточки.

### ProductList
Создан компонент, который отображает список продуктов.

Массив продуктов содержит:
- id
- name
- price

Список отображается с использованием метода map().

Каждый элемент имеет уникальный key.

### Section
Создан компонент Section, который возвращает fragment и содержит:
- заголовок (title)
- дочерние элементы (children)

Этот компонент используется для группировки других компонентов без лишних div.

## Вывод
В ходе лабораторной работы были изучены:
- создание собственных React компонентов
- передача данных через props
- использование fragments
- динамическое отображение списков
