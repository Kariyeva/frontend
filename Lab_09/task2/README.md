# Lab 09 – Virtualized Lists (React + TypeScript)

## 📌 Description
This lab demonstrates the difference between a regular list and a virtualized list using react-window.

## 🚀 Features
- Virtualized list rendering with 10,000 items
- Filtering items by text
- Comparison with regular list rendering
- Performance optimization using virtualization

## 🛠️ Technologies
- React
- TypeScript
- Vite
- react-window
