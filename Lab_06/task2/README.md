# Лабораторная работа 06 — Task 2 (React Router: динамические маршруты и query-параметры)

## Цель
Реализовать динамический маршрут `/courses/:id`, загрузку данных через `loader`,
обработку ошибок маршрута и сортировку через query-параметры.

## Что сделано
- Установлен `react-router-dom`.
- Созданы данные курсов (массив `courses`) и тип `Course`.
- Реализованы страницы и компоненты:
  - `Courses` — список курсов с ссылками на `/courses/:id`
  - `CourseDetail` — страница деталей курса по параметру `id`
  - `CourseError` — обработка ошибок загрузчика (неверный id / курс не найден)
  - `Layout`, `Home`, `About`, `NotFound` — общая структура и страницы
- Настроен `loader` (`courseLoader`), который:
  - проверяет параметр `id`
  - находит курс по id
  - выдаёт ошибку (400/404), если id неверный или курс не найден
- Реализована сортировка списка курсов через query-параметр `sort`:
  - `/courses?sort=asc`
  - `/courses?sort=desc`

## Проверка
- `/courses` открывает список курсов.
- Переход по курсу открывает `/courses/:id` и показывает детали.
- `/courses/999` (или другой несуществующий id) показывает страницу ошибки.
- Кнопка сортировки меняет `sort` в адресной строке и порядок списка.
