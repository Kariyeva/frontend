# Лабораторная работа 06 — Task 1 (React Router: базовая навигация)

## Цель
Настроить базовую навигацию в React-приложении с помощью React Router.

## Что сделано
- Установлен `react-router-dom`.
- Создан общий макет `Layout` с навигацией (Home, Courses, About), основным контентом через `<Outlet />` и футером.
- Реализованы страницы:
  - `Home` — главная страница
  - `Courses` — список курсов
  - `About` — информация о портале
  - `NotFound` — страница 404 для несуществующих маршрутов
- Настроены маршруты через `createBrowserRouter` и `RouterProvider`.

## Проверка
- Переходы по Home / Courses / About работают без перезагрузки страницы.
- При переходе на несуществующий адрес (например `/abc`) отображается страница 404.
