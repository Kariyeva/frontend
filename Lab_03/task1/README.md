# Lab 3.1 — Advanced Counter

В этом задании реализован компонент StepCounter с использованием props и state. 
Счётчик принимает initialValue и step, умеет увеличивать и уменьшать значение, 
сохраняет историю последних 5 изменений и считает количество операций. 
Также реализована кнопка Reset.

В родительском компоненте CounterApp отображаются два независимых счётчика.

Для запуска:
npm install
npm run dev
