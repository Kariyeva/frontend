# Лабораторная работа 05 — Task 1 (Типизация компонентов)

## Цель
Научиться использовать TypeScript для типизации компонентов React и их props.

## Что было сделано
- Создан файл `src/types.ts`, в котором описаны типы:
  - `User` — интерфейс для данных пользователя
  - `SkillLevel` — union type: "Beginner" | "Intermediate" | "Expert"
  - `Skill` — интерфейс для навыков

- Реализованы компоненты:

### UserCard.tsx
Компонент отображает информацию о пользователе.

Props:
- `user: User`
- `isActive?: boolean` (необязательный, по умолчанию true)
- `children: React.ReactNode`

Компонент выводит:
- имя пользователя
- email
- возраст
- дополнительный текст через children.

### SkillList.tsx
Компонент отображает список навыков.

Props:
- `skills: Skill[]`

Используется метод `.map()` для отображения списка навыков.

## Дополнительно
- В проекте используется TypeScript.
- Включён строгий режим (`strict mode`).
- Тип `any` не используется.
