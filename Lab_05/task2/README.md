## Компонент SearchApp

Реализовано приложение поиска пользователей.

### Состояние (state)
Используются типизированные состояния:

- `useState<User[]>(INITIAL_DATA)` — список пользователей
- `useState<User[]>` — отфильтрованный список
- `useState("")` — строка поиска

### Обработчики событий

- `handleSearch(event: React.ChangeEvent<HTMLInputElement>)`
  - фильтрует пользователей по имени

- `handleClear(event: React.MouseEvent<HTMLButtonElement>)`
  - очищает поле поиска
  - возвращает исходный список пользователей

### Функциональность

Приложение содержит:

- поле поиска
- кнопку Clear
- список пользователей
- сообщение **"No results found"**, если ничего не найдено

## Дополнительно
- Используется TypeScript.
- Включён строгий режим (`strict mode`).
- Тип `any` не используется.
