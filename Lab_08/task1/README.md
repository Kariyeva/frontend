# Lab 08 — Task 1 (Next.js: SSG & ISR)

## Описание
В данном задании реализовано приложение на Next.js с использованием статической генерации страниц (SSG) и инкрементальной регенерации (ISR).

## Функционал
- Главная страница (SSG)
- Список постов
- Динамические страницы постов
- ISR (обновление страницы через заданный интервал)

## Используемые технологии
- Next.js
- React
- TypeScript

## Запуск проекта

```bash
npm install
npm run dev
Открыть в браузере:

http://localhost:3000
Структура проекта

pages/index.tsx — главная страница

pages/posts/[id].tsx — динамические страницы

pages/api — API (если используется)

Особенности

Используется getStaticProps

Используется getStaticPaths

Реализована ISR (revalidate)
